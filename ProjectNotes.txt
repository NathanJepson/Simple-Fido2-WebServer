 cryptoParams: [-7, -257] = 
	-7 ECDSA w/ SHA-256	
	-257 RSASSA-PKCS1-v1_5 using SHA-256
	(both deprecated)

	-9 ECDSA using P-256 curve and SHA-256
	-19 EdDSA using Ed25519 curve
	-37 RSASSA-PSS w/ SHA-256

	If left unspecified, Chrome uses the default values of ES256 (-7) and RS256 (-257).
